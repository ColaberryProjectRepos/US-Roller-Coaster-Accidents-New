Create Dax Measure Table 13




1.    Select the Safer parks Table and right click on it and select New Measure
2.    From Power BI desktop ribbon, Home tab click on New measure 
3.    The formula bar appears along the top of the report canvas. 
4.    Fill in all the DAX formulas provided below, one at a time, and when you finish Press Enter or select Commit (checkmark icon) in the formula bar. 


5. After creating each measure choose the necessary format and decimal for each value (measure)
6. Save

DAX Formulas

1.Total Accidents = sum('safer-parks-accident-dataset'[num_injured])
2. Accidents per Month = DIVIDE('safer-parks-accident-dataset'[.Total acct],DISTINCTCOUNT(Calander[YearMonth]),0)+0
3. .Accidents MoM% = VAR VAR1= CALCULATE('safer-parks-accident-dataset'[.Total acct],DATEADD('Calander'[Date],-1,MONTH))+0 VAR VAR2= DIVIDE('safer-parks-accident-dataset'[.Total acct]-VAR1,VAR1)+0 RETURN IF((VAR2)=0,0,VAR2)
4. .LY = CALCULATE([.Total acct],DATEADD(Calander[Date],-1,YEAR))
5.  .YOY_VAR = IF([.LY]=0,0,DIVIDE(([.Total acct]-[.LY]),[.LY]))+0
6. .Accidents YoY% = AVERAGEX(VALUES(Calander[Year]),[.YOY_VAR])
7. .IND COLOR MOM = IF([.Accidents MoM%]>=0,"red","green")
 8. .IND COLOR YOY = IF([.Accidents YoY%]>=0,"red","green")
 9. .KPI IND MOM% = IF([.Accidents MoM%]<0, UNICHAR(9660),IF([.Accidents MoM%]>0, UNICHAR(9650),UNICHAR(UNICODE("-")) ))
10. .KPI IND YOY% = IF([.Accidents YoY%]<0,UNICHAR(9660),IF([.Accidents YoY%]>0,UNICHAR(9650),UNICHAR(UNICODE("-")) ))
