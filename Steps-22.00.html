Adding Visual Charts-Line and Clustered/Stacked Column 22



Line and Clustered Column Visual 1(Accidents by Month/Year)
Steps
  1. Click on an empty space on the canvas, and from the visualizations pane select Line and Clustered Column Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Tab:
  • Shared axis = Date(Year,Month,Day)
  • Column Values = Accidents Per Mth
  • Line Values = Accidents MoM, Accidents YoY
  • Tooltips = Accidents MoM, Accidents YoY
  • After filling up the above rename fields, as per gif provided below
  • Sort axis by YearMonth, and ascending order
Format Visual Tab:
